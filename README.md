# PHPD_Klausur - Media Portal

A full-stack media management application for books, films, and music built as a PHPD module exam project.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript (AJAX)
- **Backend:** PHP (OOP, MVC)
- **Database:** MySQL with PDO

## Features
- Media browsing with list and detail views
- User authentication and registration
- Comment system for registered users
- AJAX-powered search with auto-suggest
- Admin dashboard for content management
- Import/export (JSON, XML, CSV)

## Installation
1. Run `setup.bat` to initialize the database and copy files to XAMPP htdocs
2. Visit `http://localhost/gruppe1_phpd/public` in your browser

## Project Structure
- `/Backend` - PHP backend (MVC architecture)
- `/public` - Frontend assets and views
- `/Setup` - Database schema and setup scripts
- `/Documentation` - Project documentation
