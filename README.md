# PHPD_Klausur - Media Portal Application
A full-stack web application for managing and browsing books, films and music built as an exam project for the PHPD module.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript (AJAX)
- **Backend:** PHP (OOP, MVC pattern)
- **Database:** MySQL with PDO

## Features
- Books, Film and Music browsing with list and detail views
- Simple User authentication (login/registration)
- Comment system for registered users
- AJAX-powered search with auto-suggest
- Admin dashboard for content management
- Import/export media data (JSON, XML, CSV)
