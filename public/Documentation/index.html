Media Portal – Project Documentation

PHPD Module Exam Project

1. Project Overview

This project is a full-stack media management web application developed as part of the PHPD module exam. The application allows users to browse, search, and comment on different types of media, such as books, movies, and music.

Target group: End users and administrators
Purpose: Demonstrate backend, frontend, database, and MVC knowledge

2. Tech Stack

  • Frontend: HTML, CSS, JavaScript (AJAX)
  • Backend: PHP (Object-Oriented Programming, MVC architecture)
  • Database: MySQL using PDO
  • Server: Apache (XAMPP)

3. Features

  • Media browsing with list and detail views
  • User registration and authentication
  • Comment and rating system for registered users
  • AJAX-powered search with auto-suggest
  • Admin dashboard for managing content
  • Import and export of data (JSON, XML)

4. Installation Guide

  1. Install XAMPP and start Apache & MySQL
  2. Run the file setup.bat
  3. The database media_library will be created automatically
  4. Project files are copied into htdocs
  5. Open the website in your browser

6. Database Structure

The database is named media_library and contains multiple tables for users, admins, media items, comments, and contact messages.

  • users – Registered users
  • admins – Admin accounts
  • books – Book data (Google Books API based)
  • movies – Movie information
  • books_comments – Comments for books: test data have to be imported manualy
  • movies_comments – Comments for movies: test data have to be imported manualy
  • contact_messages – Messages from users

7. Project Structure

  • /Backend – PHP backend (MVC structure)
  • /public – Frontend files and views
  • /Setup – Database schema and setup scripts
